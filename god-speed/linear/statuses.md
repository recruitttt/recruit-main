# Linear Statuses

| Status | Meaning | Use When |
|---|---|---|
| Backlog | Not yet scoped | Idea exists but is not ready |
| Scoped | Accepted into sprint | The team agrees this belongs |
| Ready | Ready to start | Has enough detail and no blockers |
| In Progress | Actively being worked | Someone owns it now |
| Blocked | Cannot continue | Needs help/decision/dependency |
| Needs Review | Built but not verified | Needs review or QA |
| Verified | Passed checklist | Works as expected |
| Demo Ready | Safe for demo | Verified on demo path |
| Cut | Removed from scope | Will not ship this sprint |
