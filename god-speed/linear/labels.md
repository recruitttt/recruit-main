# Linear Labels

| Label | Use When |
|---|---|
| frontend | UI/client work |
| backend | API/server work |
| ai | AI/prompt/model work |
| data | Data/schema/seed work |
| infra | Deployment/config/devops |
| design | UX/UI/design systems |
| qa | Testing/verification |
| pitch | Pitch/demo/submission work |
| docs | Documentation |
| demo-critical | Demo breaks without this |
| stretch | Nice-to-have |
| blocked | Needs unblock |
| bug | Broken behavior |
| polish | Visual/copy polish |
| mockable | Can be safely mocked |
| cuttable | Can be removed if time is low |
| risky | Has high implementation/demo risk |
| needs-decision | Requires a decision before progress |
