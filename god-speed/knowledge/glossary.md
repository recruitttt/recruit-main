# Glossary

Define project-specific words, abbreviations, components, services, and concepts here.

| Term | Meaning | Notes |
|---|---|---|
| MVP | Smallest demoable version | Not the full product |
| Demo-critical | Required for the live demo | Must have fallback |
| Mockable | Can be faked safely/honestly | Must not overclaim |
| Cuttable | Can be removed from scope | Usually P2/P3 |
| Verified | Tested against checklist | Not just it seems done |
