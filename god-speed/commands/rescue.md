# /rescue

## Purpose

Runs final-3-hours rescue mode.

## When To Use

Use in the final 3 hours or whenever the demo path is unstable.

## Reads

- project source of truth
- issue list
- status
- demo plan
- known blockers

## Writes / Proposes

- rescue plan
- final role assignments

## Output Format

Safest demo path, what to stop building, what to cut immediately, what to mock, screenshots/videos needed, bugs worth fixing, pitch adjustments, Devpost tasks, team role assignments.

## Prompt

No new features. Pick the safest demo path. Patch only demo-breaking bugs. Record backup video/screenshots. Mark everything Demo Ready, Cut, or Known Broken.
