# /pitch

## Purpose

Writes a 3-5 minute pitch script.

## When To Use

Use after the demo plan is known and verified features are clear.

## Reads

- project source of truth
- demo plan
- verified features
- known risks

## Writes / Proposes

- pitch script

## Output Format

Opening hook, problem, solution, demo transition, product walkthrough, technical credibility, impact, honest limitations, closing line.

## Prompt

Write a 3-5 minute pitch that matches what actually works. Do not hallucinate features.
