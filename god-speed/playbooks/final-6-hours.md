# Final 6 Hours Playbook

## Rule

Reliability beats scope.

## Do Now

- Freeze non-essential work
- Identify one safe demo path
- Cut unstable P2/P3 work
- Mock risky integrations
- Create screenshots
- Record fallback video
- Finish pitch
- Finish Devpost

## Linear Status Rules

Every issue must become:

- Demo Ready
- Cut
- Known Broken
- Needs Review only if actively being verified

## What To Cut

- Auth
- Settings
- Multi-user support
- Edge cases
- Complex integrations
- Anything not visible in demo

## Required Artifacts

- Working link or video
- Demo script
- Devpost draft
- Screenshot set
- Backup plan
