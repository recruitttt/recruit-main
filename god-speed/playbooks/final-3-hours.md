# Final 3 Hours Rescue

## Rule

No new features.

## 0-45 Minutes: Pick Safest Demo Path

- Choose one working path
- Identify what can be shown live
- Identify what needs fallback
- Stop all non-demo work

## 45-90 Minutes: Patch Demo-Breaking Bugs

- Only fix bugs that block the chosen path
- Do not refactor
- Do not add new flows

## 90-135 Minutes: Backup Artifacts

- Record video
- Take screenshots
- Save seed data
- Prepare offline fallback

## 135-180 Minutes: Final Submission

- Finalize pitch
- Finalize Devpost
- Rehearse
- Submit

## Final Linear Rule

Everything is:

- Demo Ready
- Cut
- Known Broken
